# Vineyard School — Design System

## Brand
- **App Name:** Vineyard School Management System
- **Tagline:** Manage your school, one place.

## Colors
- **Primary Background:** `#0D1117` (deep charcoal/near black)
- **Secondary Background:** `#161B22` (card/surface)
- **Sidebar Background:** `#0D1117`
- **Accent (Green):** `#4ADE80` (vineyard green)
- **Accent Hover:** `#22C55E`
- **Text Primary:** `#F0F6FC`
- **Text Secondary:** `#8B949E`
- **Border:** `#30363D`
- **Danger:** `#F85149`
- **Warning:** `#E3B341`
- **Success:** `#3FB950`

## Typography
- **Font:** Poppins (Google Fonts)
- **Display:** 700 weight, large sizes for headers
- **Body:** 400/500 weight
- **Labels:** 600 weight, uppercase small caps

## Layout
- Sidebar navigation (fixed left, 240px)
- Main content area (fluid)
- Card-based content sections
- Generous padding (24px on cards)
- Table-heavy — clean dark table rows

## Components
- Dark cards with subtle border `#30363D`
- Green accent buttons
- Pill badges for status (Active/Inactive/Paid/Unpaid)
- Sidebar with icon + label per module
- Top header bar with user avatar + school name

## Mobile
- Bottom tab navigator (dark background)
- Card-heavy layout
- Same green accent, same dark palette
